Pasos para crear Repositorio:

1. www.GitHub> Crear Repositorio> Asignar Nombre
2. CMD> Dirección de la carpeta en la que se guardará> git clone (copiar el enlace del repositorio creado en GitHub)
3. CMD> Ubicarse en la dirección de la nueva carpeta creada por GitHub.
4. Agregar todos los archivos cambiados: 
	git add -A
5. Realizar commit:
	git commit -m
6. Publicar:
	git push
